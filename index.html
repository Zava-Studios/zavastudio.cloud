Lo que hacemos

Servicios especializados

Combinamos conocimientos en ciberseguridad ofensiva y defensiva con modelos de inteligencia artificial para construir soluciones que escalan.

🔍

Análisis de vulnerabilidades

Auditorías técnicas de sistemas, redes y aplicaciones web. Identificamos vectores de ataque antes de que lo haga alguien con malas intenciones. Entregamos informe ejecutivo y técnico.

Pentesting OWASP CVE
🤖

Detección de amenazas con IA

Modelos de machine learning entrenados para identificar patrones de ataque en tiempo real: anomalías de red, comportamiento malicioso y exfiltración de datos.

ML SIEM IDS/IPS
🛡️

Hardening y bastionado

Configuración segura de servidores, contenedores y entornos cloud siguiendo benchmarks CIS y NIST. Reducimos la superficie de ataque sin sacrificar operatividad.

CIS Docker Cloud
🔐

Criptografía aplicada

Implementación de protocolos criptográficos modernos: TLS, cifrado de extremo a extremo, gestión de claves y diseño de sistemas de autenticación robustos.

TLS 1.3 PKI Zero-Trust
📊

Desarrollo de modelos IA

Diseño y entrenamiento de modelos para casos de uso en seguridad: clasificación de malware, análisis forense automatizado y procesamiento de logs a gran escala.

PyTorch NLP Forense

Respuesta a incidentes

Protocolo de contención, análisis forense y recuperación ante brechas de seguridad. Documentamos el vector de ataque y proponemos medidas estructurales de mejora.

DFIR Forense Recovery

Proyectos destacados

Trabajo real, técnico y documentado. Cada proyecto nace de un problema concreto y termina con resultados medibles.

SentinelNet

Detección de intrusiones · ML
Activo

Sistema de detección de intrusiones basado en redes neuronales recurrentes. Analiza tráfico de red en tiempo real y clasifica ataques con una precisión superior al 94%. Entrenado sobre el dataset NSL-KDD y validado en entornos reales.

Python LSTM Scikit-learn Wireshark
↳ 94.3% precisión ↳ 12 tipos de ataque ↳ Tiempo real

PhishGuard

Anti-phishing · NLP
Activo

Extensión de navegador con modelo NLP para detección de páginas de phishing. Analiza URL, contenido DOM y certificados TLS para emitir un veredicto en menos de 200 ms. Integrado con feeds de inteligencia de amenazas.

BERT JavaScript Chrome API TLS
↳ <200 ms latencia ↳ 97% recall ↳ Browser extension

VaultSec

Gestión de credenciales · Cripto
En desarrollo

Gestor de contraseñas de código abierto con cifrado AES-256-GCM y derivación de clave con Argon2id. Arquitectura zero-knowledge: el servidor nunca tiene acceso a los secretos del usuario.

Rust AES-256 Argon2id Zero-Knowledge
↳ Zero-knowledge ↳ Open source ↳ Q3 2025

MalwareScope

Análisis estático · IA
En desarrollo

Plataforma de análisis estático y dinámico de binarios. Extrae características de PE headers, strings y grafos de llamadas; un clasificador entrenado sobre 80.000 muestras etiquetadas lo categoriza en familia de malware.

Python YARA GNN Sandbox
↳ 80k muestras ↳ 18 familias ↳ Sandboxed
